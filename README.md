# SupplyChainAI

SupplyChainAI is a comprehensive, AI-powered platform for modern supply chain management, built with Next.js, Genkit, and ShadCN UI. It provides a suite of tools for both administrative staff and customers to monitor, analyze, and optimize logistics and inventory.

## Key Features:

- **Admin Dashboard**: A real-time overview of the entire supply chain, featuring key metrics like total inventory, in-transit deliveries, and anomaly alerts. It includes an interactive inventory chart and a live map tracking delivery vehicles.

- **AI-Powered Forecasting**: Leverages historical data to predict future product demand, helping to prevent stockouts and reduce overstock. It provides actionable recommendations and a seamless transition to placing new orders.

- **AI Operations Analyst**: Monitors a live stream of supply chain events, automatically identifying and summarizing anomalies (e.g., delays, stock issues) and suggesting corrective actions.

- **Agentic Logistics Optimization**: An intelligent agent that optimizes delivery routes by analyzing traffic, distance, and other factors. It supports human-in-the-loop workflows, allowing for manual overrides and confirmation before finalizing shipments.

- **Customer Panel**: A dedicated portal for customers to manage their inventory, view stock levels, and initiate demand forecasts for their products when stock is low, seamlessly connecting them to the logistics workflow.
